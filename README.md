# pytest-qfield

A [pytest](https://docs.pytest.org) plugin for testing [QField](https://github.com/opengisch/QField) QML plugins.

> [!CAUTION]
> Under active development! API not stable yet.


## Inspirations
- [pytest-qgis](https://github.com/osgeosuomi/pytest-qgis)
- [pytest-qt](https://github.com/pytest-dev/pytest-qt)
- [This Stackoverflow answer](https://stackoverflow.com/a/71832084)
